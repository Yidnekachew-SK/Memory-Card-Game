# Memory-Card-Game

React based memory card game from The Odin project course where users try to click all the cards once.  
The game uses [PokéAPI](https://pokeapi.co/) to fetch the images and names shown on the cards.

**`rule:`** Don't click the card twice!

[Live Demo](https://yidnekachewsk-memory-card-game.netlify.app/)
